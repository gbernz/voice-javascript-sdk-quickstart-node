Bob's Scuba Shop Example - Twilio Voice JavaScript SDK

Call Flow:

  1. End user calls Bob's Scuba Shop (1st Twilio Number)
  2. Twilio number webhooks to Bob's server (/voice endpoint) and gets instructions to enqueue the call
  3. Status callback hits (/delivery-callback endpoint) to update frontend
  4. Bob clicks "Call next in queue" to dial 2nd Twilio number
  5. Twilio number webhooks to Bob's server (/voice endpoint) and gets instructions to queue next call
  6. Queue verb includes instructions prior to bridging connections at (/about-to-connect)
  7. Status callback hits (/delivery-callback endpoint) to update frontend
